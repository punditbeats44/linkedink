Chrome Extension — User Guide & Setup
What is LinkedInk?
LinkedInk is a Chrome extension that uses Claude AI to write LinkedIn content for you in seconds. Instead of staring at a blank text box, simply describe what you want — and LinkedInk writes it.

What it can write for you:
•	📝  LinkedIn Posts — with tone and length control
•	💬  Comments — thoughtful replies to any post
•	🤝  Connection Requests — personalized outreach messages
•	💼  Job Application Messages — tailored to each role

Requirements
LinkedInk uses the Anthropic Claude AI API. You need a free Anthropic account and API key to use it. API usage costs fractions of a cent per generation.

•	Google Chrome browser (version 88 or higher)
•	An Anthropic API key (free to get — instructions below)
•	Internet connection

Step-by-Step Setup
Step 1: Get Your Anthropic API Key
1.	Go to console.anthropic.com in your browser
2.	Click "Sign Up" and create a free account
3.	After logging in, click "API Keys" in the left menu
4.	Click "Create Key" — give it any name (e.g. "LinkedInk")
5.	Copy the key that appears (starts with sk-ant-...) — save it somewhere safe
6.	Add a payment method under Billing (you get $5 free credits — enough for hundreds of posts)

Step 2: Install the Extension
7.	Download and unzip the LinkedInk extension file
8.	Open Google Chrome and go to: chrome://extensions/
9.	Toggle "Developer Mode" ON (switch in the top-right corner)
10.	Click "Load unpacked" button
11.	Select the unzipped LinkedInk folder
12.	The LinkedInk icon (Li) will appear in your Chrome toolbar

Step 3: Activate LinkedInk
13.	Click the LinkedInk icon in your Chrome toolbar
14.	Paste your Anthropic API key in the box
15.	Click "Activate LinkedInk"
16.	You're ready to go!

How to Use Each Feature
📝 Post Generator
Perfect for creating engaging LinkedIn posts that get real engagement.
•	Click the "Post" tab
•	Describe what your post is about in the text box
•	Choose your tone: Professional, Storytelling, Inspirational, Casual, or Bold
•	Choose length: Short, Medium, or Long
•	Click "Generate Post"
•	Copy the result or regenerate for a different version

💬 Comment Writer
Write thoughtful comments that add real value to conversations.
•	Click the "Comment" tab
•	Paste the LinkedIn post you want to comment on
•	Optionally describe your angle or reaction
•	Choose a tone: Thoughtful, Agreeable, Challenging, or Supportive
•	Click "Generate Comment"

🤝 Connection Request
Send personalized connection requests that actually get accepted.
•	Click the "Connect" tab
•	Enter the person's name and title
•	Explain why you want to connect
•	Add one line about your background
•	Click "Generate Message" — keeps it under LinkedIn's 300 character limit

💼 Job Application
Write confident, tailored messages for job applications.
•	Click the "Job Apply" tab
•	Enter the job title and company name
•	Paste key requirements from the job description
•	List your top relevant strengths
•	Click "Generate Application Message"

Tips for Best Results
•	Be specific in your prompts — the more detail you give, the better the output
•	For posts: describe a real story or insight, not just a topic (e.g. "I just failed my first startup and here's what I learned" works better than "entrepreneurship")
•	For connection requests: always mention something specific about the person
•	For job applications: paste the actual requirements from the job description
•	Use Regenerate if the first result isn't quite right — each generation is different

Troubleshooting
Problem	Solution
'Invalid API Key' error	Make sure your key starts with sk-ant- and was copied fully
No response / loading forever	Check your internet connection and try again
'Insufficient credits' error	Add a payment method at console.anthropic.com/billing
Extension not showing in toolbar	Click the puzzle icon in Chrome toolbar and pin LinkedInk
Output is too generic	Add more specific details to your prompt

Privacy & Security
•	Your API key is stored locally on your device only — it is never sent to any third party
•	Your prompts are sent directly to Anthropic's API (the same company that makes Claude AI)
•	No account or login is required for LinkedInk itself
•	You can delete your API key at any time from chrome://extensions/ → LinkedInk → Storage
